# Summary

This is an object of constant width that I made based off a Numberphile video that I saw, as well as a video done by Maker's Muse. They are pretty cool and oddly satisfying to play around with.

Here is a cool Numberphile video over them:
https://www.youtube.com/watch?v=cUCSSJwO3GU

# Print Settings

Printer: HICTOP i3
Rafts: Doesn't Matter
Supports: Yes
Resolution: .01
Infill: 15%

Notes: 
I found that .1mm layer heights made a smoother surface on the bottom of the print which makes it roll a lot cleaner.
