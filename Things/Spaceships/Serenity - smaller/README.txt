# Summary

This is an accurate model of the Firefly class transport ship from the Serenity movie, with some modification to improve FDM printing.

Attached test print is about 50% scale. Any smaller and you lose pipe detail. 

I'm currently working on printing a high quality split version with limited need for support. Will post when I get around to gluing.
